Guía de configuración de proyecto Chat GPT para Score Viewer

3 pasos simples para montar tu proyecto en ChatGPT y usar music21 sin liarte.

1) Archivos

Sube estos dos ficheros en la sección Archivos de tu proyecto:

  • PROMPT_CHATGPT_FINAL.md → reglas detalladas.
  • music21_rules.json → banderas y validación.

2) Instrucciones del proyecto

Pega este bloque en Configuración → Instrucciones. Es lo que ChatGPT verá siempre:

Rol: Generador de ejercicios y composiciones con music21 para app Score Viewer.
Obedece SIEMPRE las reglas de PROMPT_CHATGPT_FINAL.md y music21_rules.json.

Reglas clave:
1) Sin bucles ni listas de comprensión.
2) IDs únicos en formato {tipo}-m{compás}-{ordinal}.
3) Textos siempre en offset 0; ChordSymbols con offsets solo si hay varios por compás.
4) TimeSignature SOLO en el primer compás de cada parte.
5) Usar Unicode ♭♯♮ en lyrics/intervalos.
6) Título obligatorio y terminar con `score = s`.
7) Si incumples alguna regla → devuelve lista de errores + versión corregida.

3) Mensaje semilla

Al abrir un chat nuevo dentro del proyecto, envía este mensaje como primer paso:

Primero, resume en 6 puntos las reglas que has aprendido de PROMPT_CHATGPT_FINAL.md y music21_rules.json.
Después, genera un ejemplo mínimo con music21:
- Un compás de Cmaj7 en 4/4
- 4 negras (C4, D4, E4, F4)
- Incluir título, clave, armadura, tempo con id, ChordSymbol con id, y terminar con `score = s`.
Debe cumplir TODAS las reglas (sin bucles, IDs únicos, textos en offset 0, timeSignature solo en el primer compás, Unicode ♭♯♮ si procede).
Si incumples alguna regla, primero devuelve lista de errores y luego la versión corregida.
Además de las reglas de notación, consulta las guías de composición clásica y moderna en Documentos para aplicar las reglas estilísticas.

4) Flujo de uso

  1. Configura: sube archivos y pega instrucciones.
  2. Inicia: usa el mensaje semilla una sola vez al empezar un chat.
  3. Trabaja: pide ejercicios o composiciones; no repitas el semilla salvo que abras otro chat nuevo.
